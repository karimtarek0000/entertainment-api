Available Endpoints

  • GET /api/users - Get all users
  • GET /api/users/:id - Get user by ID
  • GET /api/movies - Get all movies
  • GET /api/movies/:id - Get movie by ID
  • GET /api/tvSeries - Get all TV series
  • GET /api/tvSeries/:id - Get TV series by ID

Query Parameters

  • Filtering: ?category=trending, ?year=2024
  • Sorting: ?_sort=year&_order=desc
  • Pagination: ?_page=1&_limit=10

Example Requests

  • /api/movies
  • /api/movies?category=trending
  • /api/users
  • /api/tvSeries
